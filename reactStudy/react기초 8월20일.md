## JSX
Javascript 와 HTML의 합성어로 XML사용을 허용하는 JS의 확장문법
JSX를 사용하면 어플리케이션 속도를 높히고 가독성을 높혀 코드작성을 더 쉽고 빠르게 할수있음.
